#  교대로
> 학교 앞에 있는 음식점과 카페의 메뉴 및 잔여석를 알려주는 웹사이트입니다.

## 프로젝트 목적
- 기존 어플등이나 웹에는 등록되어 있지 않은 가게가 많이 있음
- 저녁 시간에 음식점이나 시험기간 카페에 갈 때 자리가 없어 학생들이 헛걸음을 하지 않기 위함

## 프로젝트 설명
- 각 가게의 이름, 전화번호, 주소, 메뉴 등의 정보를 소개
- 회원 유형에 맞는 마이페이지를 구성

## 설치
- Visual Studio Code
- Mysql
- node js 설치

```
npm install express, bodyParser, express-session, mysql, ejs, request, session-file-store
```


## 기능
> 회원가입 시 일반 회원과 사업자 회원으로 나눠서 가입할 수 있다.
> 일반 사용자는 음식점과 카페들의 메뉴가 가게 정보를 볼 수 있다.
> 사업자는 본인의 가게에 메뉴 수정 및 삭제가 가능하고 총 좌석과 잔여 좌석을 수정할 수 있으며 새로운 가게를 추가할 수 있다.
> 관리자는 사업자가 요청한 가게에 대해 승인하여 웹 사이트에서 보여질 수 있게 할 수 있다.
